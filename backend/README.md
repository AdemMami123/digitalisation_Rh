# Backend API

API REST pour la plateforme de gestion RH.

## Routes disponibles

### Health Check
- `GET /` - Message de bienvenue
- `GET /api/health` - Vérification du statut de l'API

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Variables d'environnement

Voir `.env.example` pour la configuration requise.
